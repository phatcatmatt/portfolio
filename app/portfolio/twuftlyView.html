twuft.ly

minimalist Twitter analytics

Built with HTML5, CSS3, Angular, MongoDB, Express, Node, sessions, D3, and Twitter's API.

Displays the number of 'favorites' and 'retweets' per tweet for any public Twitter user as well as a graph showing the user's follower count over time.

Each Twitter user searched on the site is stored in my database where the server requests an updated follower count from Twitter on a regular interval. This information is stored again in the database and presented as a D3 chart. This allows the user to correlate any follower increase or decrease to a given tweet or time period and analyze trends for any public account.

Won runner-up for "Most Technical App".
